# Запуск проекта бота в Node-RED

Запускаем контейнер с node-red:
```docker-compose up```

Интерфейс node-red будет доступен локально: [http://127.0.0.1:11880/](http://127.0.0.1:11880/)

Сразу после запуска для работы будет нехватать модулей для работы.
Список зависимостей можно посмотреть в node_red/package.json
Нужно поставить эти модули (Главное меню -> Управление палитрой -> Установить).

При разработке, чтобы не было конфликтов между копиями бота, нужно зарегестрировать своего бота у `@botfather`
Полученные токен и имя бота задать в конфиге ноды `Telegram Receiver` (Свойства -> Bot configuration) и указать этот же конфиг у ноды `Telegram Sender`

 `Развернуть` потоки.
